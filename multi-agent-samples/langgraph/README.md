# LangGraph Multi-Agent Examples

This folder contains multi-agent implementations using LangGraph, an open-source framework for building stateful, multi-actor applications with LLMs.

## Setup

1. Create and activate a virtual environment:

```bash
python3 -m venv .venv
source .venv/bin/activate
```

2. Install dependencies:

```bash
pip install -e .
```

Or with development dependencies:

```bash
pip install -e ".[dev]"
```

## Structure

- `pyproject.toml`: Project configuration and dependencies
- Additional examples and implementations to be added

## Requirements

- Python 3.10 or higher
- LangGraph
- LangChain and LangChain OpenAI
- OpenAI API key (set via environment variable `OPENAI_API_KEY`)
